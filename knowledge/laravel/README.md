# DevSplit Laravel Engineering Handbook

This handbook defines how DevSplit builds Laravel applications.

## Default Architecture
Controller → Action → Service → Model

## Defaults
- MariaDB by default
- PostgreSQL when justified
- Docker required
- PHPUnit
- REST or GraphQL depending on project
- Vue / Livewire / React depending on project
- Repositories: rarely
